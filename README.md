# RSpec

A BDD-flavored Ruby testing framework

Let's get started: http://rspec.info/

### GemFile
source 'https://rubygems.org'

gem 'rspec', '~> 3.0'


bundle install --binstubs

bin/rspec --init

gem search -l rspec

### Execute tests
rspec --format doc --color
